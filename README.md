Built using:

- Front-End : React
- Back-End : NodeJs, ExpressJs, MySql
  
In the /Note-Vault/client, run:
### `npm start`

In the /Note-Vault/server, run:
### `nodemon server.js`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
